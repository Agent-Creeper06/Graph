# Graph

## Общее описание
Инструмент для визуализации графа зависимостей пакетов npm

## Функции и настройки

### Параметры командной строки:
- `--package`: Имя анализируемого пакета (обязательный)
- `--repo`: URL репозитория или путь к тестовому файлу (обязательный)
- `--test`: Включить режим тестирования
- `--max-depth`: Максимальная глубина анализа (по умолчанию: 5)
- `--filter`: Подстрока для фильтрации пакетов

### Основные функции:
1. **Анализ зависимостей**: Рекурсивный обход графа зависимостей с помощью DFS
2. **Обнаружение циклов**: Выявление циклических зависимостей
3. **Обратные зависимости**: Поиск пакетов, зависящих от целевого пакета
4. **Визуализация**: Генерация графического представления графа

### Ввод команд
```bash
Необходимо для работы:
`--package` имя покета `--repo` имя репозитория `--test``
Дополнительные команды:
`--max-depth` число
`--filter` имя репозитория
```
